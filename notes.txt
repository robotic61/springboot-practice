1. Database

1)Create a new database(for project):
mysql command:
mysql -u root -p
enter
CREATE DATABASE first_springboot;
SHOW DATABASES;
USE first_springboot;
SHOW TABLES;

this creates database with NO TABLE.

Then:
Spring Boot (JPA) will create tables from your @Entity

2) configure the application.properties to include the database details
spring.jpa.hibernate.ddl-auto=update
Spring Boot checks your Entity.
If table does not exist, create it.
If fields change, update table.

spring.jpa.show-sql=true -> prints SQL queries in console
spring.jpa.properties.hibernate.format_sql=true -> make it readable

3) Add MySQL driver in pom.xml:

JPA = brain 🧠 (knows how to talk to DB)
Driver = cable 🔌 (actual connection)

<dependency>
    <groupId>com.mysql</groupId>
    <artifactId>mysql-connector-j</artifactId>
    <scope>runtime</scope>
</dependency>

JSON → (setters) → Object → (getters) → Database
Database → (setters) → Object → (getters) → JSON

Setters → fill object
Getters → read object
save() → uses getters
find() → uses setters
🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴🔴
When findAll() runs, Spring uses setters to fill the object with data from each row
Then Objects in the list are converted to JSON using getters.

Spring uses getters to read the data from your object when saving to the database

save() → add row
findAll() → get all
findById() → get one
deleteById() → remove one
